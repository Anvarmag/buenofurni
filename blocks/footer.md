# Блок: Footer (Подвал сайта) — BUENOFURNI

## 1. Цель блока
Служить финальным навигационным и функциональным узлом страницы. Подвал предоставляет быстрый доступ к основным разделам каталога, контактной информации, социальным сетям и обязательным юридическим документам. Дизайн должен быть чистым, упорядоченным и поддерживать премиальный стиль бренда (воздушность, эстетика, отсутствие лишнего шума).

## 2. Структура (DOM Blueprint)
*   `<footer>`
    *   **Брендинг:** Логотип "BUENOFURNI".
    *   **Навигация (Меню ссылок):**
        *   `[Каталог](/catalog)`
        *   `[На заказ](/custom)`
        *   `[Материалы](/materials)`
        *   `[Производство](/production)`
        *   `[Контакты](/contacts)`
    *   **Контакты (Contact Placeholders):**
        *   Телефон: `<a href="tel:+7...">[Телефон для связи]</a>`
        *   Email: `<a href="mailto:...">[Электронная почта]</a>`
        *   Город/Адрес: `[Город, Улица, Шоурум]`
    *   **Социальные сети (Icons/Links Placholders):**
        *   Иконка WhatsApp (ведет на ссылку `wa.me/...`)
        *   Иконка Telegram (ведет на ссылку `t.me/...`)
    *   **Нижний колонтитул (Bottom Bar):**
        *   **Юридические ссылки (Legal links):**
            *   `[Политика конфиденциальности](/privacy-policy)`
            *   `[Пользовательское соглашение](/user-agreement)`
        *   **Copyright:** `© BUENOFURNI, 2026`

## 3. Адаптив (Grid Layout & Mobile-first)
*   **Mobile (Меньше `768px` - Mobile-first UX):**
    *   Макет "1 колонка" (1 column vertical stack). 
    *   Элементы (Логотип, Навигация, Контакты, Соцсети) выстраиваются строго друг под другом по левому краю или по центру.
    *   **Large tap targets:** Ссылки меню и юридические ссылки на смартфонах должны иметь высоту/ширину не менее `44px` и достаточные отступы (например, `padding-block: 10px; margin-bottom: 8px`), чтобы исключить случайные нажатия соседних ссылок.
*   **Tablet (`768px` - `1024px`):**
    *   Макет "2 колонки" (2 columns grid).
    *   Например: Слева – Логотип и Навигация. Справа – Контакты и Соцсети.
    *   Нижний колонтитул (юридические ссылки и копирайт) остается растянутым на всю ширину под тонким разделителем (Divider).
*   **Desktop (`1024px` и выше):**
    *   Широкая сетка (3 или 4 колонки: Брендинг / Навигация / Контакты / Соцсети).
    *   Нижний ряд с политикой ложится горизонтально в одну линию с копирайтом.

## 4. Accessibility notes (Заметки по доступности)
*   **Семантика HTML:** Весь блок оборачивается в семантический тег `<footer role="contentinfo">`.
*   **Навигация:** Группа главных ссылок должна быть завернута в список `<ul><li><a...>...</li></ul>` и обернута тегом `<nav aria-label="Нижняя навигация">`.
*   **Скрытые метки:** Все иконки социальных сетей (SVG Telegram/WhatsApp) обязаны иметь текст для скринридеров, реализованный через `aria-label="Написать в Telegram/WhatsApp"` в родительском теге `<a>`.
*   **Контрастность:** Серый текст вторичных ссылок (например, политики) не должен сливаться с фоном подвала (Соблюдение Contrast Ratio не ниже 4.5:1 для нормального текста по WCAG 2.1).
